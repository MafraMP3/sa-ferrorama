1. ## **INTRODUÇÃO** {#introdução}

No desenvolvimento existem muitas ferramentas para que tudo funcione perfeitamente e umas das ferramentas que é necessário é o PHP e CRUD, esse dois caminham juntos para criar sites dinâmicos e interativos. 

O PHP é uma das linguagens mais essenciais para a web por ser versátil, robusta e amplamente adotada ele é ótimo para criar sites dinâmicos e bom para que os desenvolvedores criem rapidamente aplicações que interagem com o servidor e bancos de dados, indo além de simples páginas HTML

 Já o CRUD é a base de tudo, pois ele vai representar todas as quatros operações básicas dos bancos de dados. Ele é o responsável por toda a interação de inserção, leitura, atualização e exclusão de informações, ele se torna essencial para qualquer tipo de tela de cadastro de alunos, vagas de empregos e muito mais 

2. ## **O QUE É PHP**  {#o-que-é-php}

PHP é uma linguagem de programação voltada para principalmente no desenvolvimento web. Ele é ótimo para criar sites dinâmicos permitindo a interação entre usuário e servidor e o banco de dados

	Essa linguagem trabalha junto com servidor ou seja antes da página ser enviada ao usuário o PHP faz várias outras funções como validar formulários, fazer login de usuários, cadastrar informações em bancos de dados e gerar páginas personalizadas. 

3. ## **O QUE É CRUD** {#o-que-é-crud}

	CRUD é um conjunto de operações que são utilizadas em sistemas e bancos de dados para gerenciar informações. O nome CRUD é formado pelas iniciais das palavras em inglês: Create, Read, Update e Delete, que significam criar, ler, atualizar e excluir.  
Essas operações são essenciais para o funcionamento de praticamente qualquer sistema que trabalhe com armazenamento de dados. Por meio do CRUD, é possível cadastrar novas informações, visualizar registros já existentes, alterar dados e remover informações quando necessário. 

### 

### **3.1 Create** {#3.1-create}

Essa operação é responsável para inserir novas informações no banco de dados ou seja um informação que usuário dá o banco de dados guarda

### **3.2 Read** {#3.2-read}

Essa operação tem a função de consultar informações armazenadas no banco de dados. Assim por meio dela os usuários podem acessar os dados já cadastrados

### **3.3 Update** {#3.3-update}

A operação Update é utilizada para modificar informações já existentes no banco de dados. Ela permite corrigir ou atualizar dados sempre que necessário. 

### **3.4 Delete** {#3.4-delete}

A operação Delete é responsável por remover informações do banco de dados. Essa operação é utilizada quando determinados dados não são mais necessários no sistema.
