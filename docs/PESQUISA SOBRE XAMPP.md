**PESQUISA SOBRE XAMPP**

**O que é XAMPP?**  
O XAMPP é um pacote com os principais softwares de código aberto do utilizado para simular condições de servidores reais, criando um servidor web local na sua maquina, suas ferramentas incluem FTP, banco de dados MySQL e Apache com suporte às linguagens PHP e Perl.

**X \- Multiplataforma :**  
	O “X” que inicia o acrônimo faz referência a sua capacidade de ser aplicado em diferentes plataformas, identificando que o mesmo funciona de forma igual em sistemas Windows, Linux e macOS.

**A \-  Apache :**  
Apache é um dos servidores web mais conhecido no mundo, e ele é de extrema importância para servir páginas web aos navegadores. Ele interpreta e executa arquivos como HTML e PHP para responder as requisições dos usuários quando entram no site. Com o Apache vai ser possível fazer simulações de servidor real no seu computador assim podendo realizar teste do site

**M \- MariaDB/MySQL :**   
	O MariaDB é o sistema de gerenciamento de banco de dados utilizado pelo XAMPP para simular um Banco de Dados localmente, assim possibilitando a criação, configuração e utilização dele no aplicativo

**P \- PHP :**  
	O PHP é a linguagem de programação usada no xampp dentro do html para gerir os códigos e scripts que serão rodados, especialmente os códigos do lado do servidor.

**P \- Perl :**   
Perl funciona como o interpretador de códigos do XAMPP escritos em perl. É uma linguagem de programação poderosa e funciona como uma base de PHP.

**Instalação**  
Para baixar o XAMPP primeiramente deve-se acessar o site oficial da Apache Friends e faça o download da versão do XAMPP para seu Sistema Operacional. Depois de baixar, execute o arquivo.  
Seleção de componentes: por padrão todos os atributos já são selecionados, caso veja algo que você não deseja instalar, desmarque a caixa de seleção ao lado.   
Selecione a localização da pasta, a linguagem e ao continuar o download irá começar. Ao terminar o download clique em ‘finish’ e o programa vai abrir.

**Importância do XAMPP para Desenvolvimento Local**   
O XAMPP é importante porque permite que desenvolvedores criem e testem aplicações web diretamente no próprio computador. Não precisa de internet para testar projetos, ambiente seguro para aprendizado, facilidade de instalação, compatibilidade com diversas tecnologias web, permite identificar erros antes da publicação do site.
